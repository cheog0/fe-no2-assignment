## 📝 구현 요약

- **상태 관리**: Redux Toolkit으로 중앙 집중형 상태 관리 구현
- **알림**: react-toastify로 모든 사용자 메시지 처리
- **디자인 시스템**: styled-components로 일관된 UI 구성 및 반응형 디자인 적용
- **라우팅**: `react-router-dom`을 통해 Home, Dex, Detail 페이지 이동 완성
- **기능 완료**: 필수 및 도전 기능 전체 구현 완료
